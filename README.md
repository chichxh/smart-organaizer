# Smart Organaizer
ИИ ассистент, готовый управлять вашим временем. Интегрирован с календарем Google для создания событий

Smart Organizer помогает автоматизировать планирование: вы задаёте задачи, идеи или запросы — он находит свободные интервалы в календаре и создаёт события.
Работает локально (на вашем компьютере), использует токен от API GigaChat (или аналогичного) и доступ к Google Calendar.

### Локальный запуск
#### 1. Инициализируйте среду и зависимости  
poetry init  
poetry install  

#### 2. Создайте файл `.env` и добавьте:
GIGACHAT_TOKEN=<ваш токен>

#### 3. Создайте файл `credentials.json` с ключом доступа к Google Calendar API (OAuth-2).

#### 4. Запустите:  
poetry run python main.py

### Возможности
- Создание событий на основании текста (например: “Запланируй встречу с Иваном завтра в 15:00 на 30 мин”)
- Поиск свободного времени в календаре
- Автоматическая вставка события с описанием, участниками, временем
- Локальный запуск: никаких облачных серверов — всё работает у вас
